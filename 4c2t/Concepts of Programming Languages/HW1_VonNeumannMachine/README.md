# Машина фон Неймана

Инструкция -- 6 байт.
Пример: 12 34 56 78 9A BC
12 -- номер команды
34 -- флаги команды (см. команду print)
56 78 9A BC -- значение (4 байта ~ int) или адрес

Инструкции:
mov addr1 addr2

Устройство образа программы:
1. ip
2. sp
3. arithmetic result
4. glob_1
...
n+3. glob_n

n+4. instr_1
...
n+m+3. instr_m

n+m+4. stack_1
...
n+m+k+3. stack_k
